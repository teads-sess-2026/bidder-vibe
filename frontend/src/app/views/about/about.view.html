About

This is the starter dashboard for the Teads 2026 Software Engineering Summer School.

It connects to a bidder's Stats API and shows its performance: an overview of headline metrics, a per-creative breakdown, targeting results, and recent trends.

Point the dashboard at your own bidder by changing backendBaseUrl in src/environments/environment.development.ts (local development) or src/environments/environment.ts (production build).

The API is documented here.
